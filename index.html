NAME

lo-kiss - or lokicalmito in other places

SYNOPSIS

lo-kiss [-minimalism] [-art 3D, fiction] [-simple software]

DESCRIPTION

-minimalism

I ask myself after a while of using/owning a thing "How much have I used this item/software?" If my very in-depth analysis turns out that I rarely use that, then I am going to get rid of it.

-simple software

I build software that I find simple. The 'kiss' in my name stands for 'Keep it simple, stupid' calling you and myself the stupid.
